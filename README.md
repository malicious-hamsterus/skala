# js env starter

[![Build status](https://ci.appveyor.com/api/projects/status/pgje30jtk2oevnyy/branch/master?svg=true)](https://ci.appveyor.com/project/homutovan/starter/branch/master)

### Installation

```
npm install
```

### Start Dev Server

```
npm start
```

### Build Prod Version

```
npm run build
```

### Run eslint

```
npm run lint
```

### Run tests

```
npm run test
```

### Run coverage

```
npm run coverage
```
